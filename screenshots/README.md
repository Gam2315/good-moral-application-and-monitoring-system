## üìù How to Take Good Screenshots

1. **Full Page Captures**: Use browser extensions like "Full Page Screen Capture"
2. **Clean Data**: Use demo/test data, avoid real student information
3. **Consistent Size**: Try to keep screenshots at similar resolutions (1920x1080 or 1366x768)
4. **Highlight Features**: Consider adding annotations for important features
5. **File Naming**: Use descriptive, lowercase names with hyphens
